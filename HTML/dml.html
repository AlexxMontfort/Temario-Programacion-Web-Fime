Lenguaje de Manipulación de Datos (DML)

¿Qué es DML?

DML, o Lenguaje de Manipulación de Datos, es una parte del lenguaje SQL que se utiliza para interactuar y manipular los datos almacenados en las bases de datos relacionales. A través de comandos DML, los usuarios pueden realizar operaciones CRUD (Crear, Leer, Actualizar y Borrar) en los datos.

Principales Comandos DML

INSERT

El comando INSERT se utiliza para añadir nuevos registros a una tabla.

INSERT INTO nombre_tabla (columna1, columna2, columna3, ...)
VALUES (valor1, valor2, valor3, ...);

Ejemplo:

INSERT INTO empleados (nombre, edad, departamento)
VALUES ('Juan Pérez', 30, 'Ventas');

SELECT

El comando SELECT se utiliza para recuperar datos de una o más tablas. Es el comando más utilizado en SQL.

SELECT columna1, columna2, columna3, ...
FROM nombre_tabla
WHERE condición;

Ejemplo:

SELECT nombre, edad
FROM empleados
WHERE departamento = 'Ventas';

UPDATE

El comando UPDATE se utiliza para modificar registros existentes en una tabla.

UPDATE nombre_tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condición;

Ejemplo:

UPDATE empleados
SET edad = 31
WHERE nombre = 'Juan Pérez';

DELETE

El comando DELETE se utiliza para eliminar registros de una tabla.

DELETE FROM nombre_tabla
WHERE condición;

Ejemplo:

DELETE FROM empleados
WHERE nombre = 'Juan Pérez';

Lenguaje de Definición de Datos (DDL)

¿Qué es DDL?

DDL, o Lenguaje de Definición de Datos, es una parte del lenguaje SQL que se utiliza para definir y gestionar la estructura de la base de datos, incluyendo la creación, modificación y eliminación de esquemas, tablas, índices, vistas y otros objetos de la base de datos.

Principales Comandos DDL

CREATE

El comando CREATE se utiliza para crear nuevos objetos en la base de datos, como tablas, vistas e índices.

Crear una Tabla

CREATE TABLE nombre_tabla (
    columna1 tipo_dato [constraints],
    columna2 tipo_dato [constraints],
    ...
);

Ejemplo:

CREATE TABLE empleados (
    id INT PRIMARY KEY,
    nombre VARCHAR(100),
    edad INT,
    departamento VARCHAR(50)
);

Crear una Vista

CREATE VIEW nombre_vista AS
SELECT columnas
FROM nombre_tabla
WHERE condición;

Ejemplo:

CREATE VIEW vista_empleados_ventas AS
SELECT nombre, edad
FROM empleados
WHERE departamento = 'Ventas';

ALTER

El comando ALTER se utiliza para modificar la estructura de los objetos existentes en la base de datos.

Modificar una Tabla

ALTER TABLE nombre_tabla
ADD columna tipo_dato [constraints];

ALTER TABLE nombre_tabla
DROP COLUMN nombre_columna;

ALTER TABLE nombre_tabla
MODIFY COLUMN nombre_columna tipo_dato;

Ejemplo: Agregar una columna

ALTER TABLE empleados
ADD salario DECIMAL(10, 2);

Ejemplo: Eliminar una columna

ALTER TABLE empleados
DROP COLUMN edad;

Ejemplo: Modificar una columna

ALTER TABLE empleados
MODIFY COLUMN nombre VARCHAR(150);

DROP

El comando DROP se utiliza para eliminar objetos de la base de datos.

Eliminar una Tabla

DROP TABLE nombre_tabla;

Ejemplo:

DROP TABLE empleados;

Eliminar una Vista

DROP VIEW nombre_vista;

Ejemplo:

DROP VIEW vista_empleados_ventas;

TRUNCATE

El comando TRUNCATE se utiliza para eliminar todos los registros de una tabla, pero mantiene la estructura de la tabla.

TRUNCATE TABLE nombre_tabla;

Ejemplo:

TRUNCATE TABLE empleados;

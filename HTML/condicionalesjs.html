Condicionales en JavaScript

Condicionales if y else

La sentencia if se utiliza para ejecutar un bloque de código solo si una condición especificada es verdadera. La sentencia else puede usarse para ejecutar otro bloque de código si la condición del if es falsa.

let edad = 18;

if (edad >= 18) {
    console.log("Eres mayor de edad.");
} else {
    console.log("Eres menor de edad.");
}

Condicional else if

La sentencia else if se utiliza para especificar una nueva condición si la condición anterior es falsa.

let hora = 14;

if (hora < 12) {
    console.log("Buenos días");
} else if (hora < 18) {
    console.log("Buenas tardes");
} else {
    console.log("Buenas noches");
}

Condicional switch

La sentencia switch evalúa una expresión y ejecuta el bloque de código correspondiente a la primera coincidencia del caso (case). La sentencia break se usa para salir del bloque switch después de ejecutar un caso específico.

let dia = 3;

switch (dia) {
    case 1:
        console.log("Lunes");
        break;
    case 2:
        console.log("Martes");
        break;
    case 3:
        console.log("Miércoles");
        break;
    case 4:
        console.log("Jueves");
        break;
    case 5:
        console.log("Viernes");
        break;
    default:
        console.log("Fin de semana");
}

Operador Ternario

El operador ternario es una forma concisa de realizar una declaración condicional. Tiene tres partes: una condición seguida de un signo de interrogación (?), una expresión a ejecutar si la condición es verdadera, y una expresión a ejecutar si la condición es falsa, separadas por dos puntos (:).

let edad = 20;
let mensaje = (edad >= 18) ? "Eres mayor de edad." : "Eres menor de edad.";
console.log(mensaje);

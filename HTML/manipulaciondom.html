Manipulación del DOM en JavaScript

Acceso a Elementos del DOM

Para manipular el DOM, primero necesitas seleccionar los elementos que deseas modificar. Hay varias formas de acceder a los elementos del DOM:

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll

getElementById

Selecciona un elemento por su atributo id.

let elemento = document.getElementById('miId');

getElementsByClassName

Selecciona todos los elementos con una clase específica. Devuelve una colección HTML.

let elementos = document.getElementsByClassName('miClase');

getElementsByTagName

Selecciona todos los elementos de un tipo específico de etiqueta. Devuelve una colección HTML.

let elementos = document.getElementsByTagName('div');

querySelector

Selecciona el primer elemento que coincide con un selector CSS.

let elemento = document.querySelector('.miClase');

querySelectorAll

Selecciona todos los elementos que coinciden con un selector CSS. Devuelve una NodeList.

let elementos = document.querySelectorAll('.miClase');

Modificación del Contenido

Una vez que hayas seleccionado un elemento, puedes modificar su contenido utilizando propiedades como innerHTML, innerText y textContent.

let elemento = document.getElementById('miId');
elemento.innerHTML = 'Nuevo contenido';

Modificación de Atributos

Puedes modificar los atributos de un elemento utilizando métodos como setAttribute y getAttribute.

let elemento = document.getElementById('miId');
elemento.setAttribute('class', 'nuevaClase');

Estilo de Elementos

Puedes modificar los estilos CSS de un elemento utilizando la propiedad style.

let elemento = document.getElementById('miId');
elemento.style.color = 'red';
elemento.style.backgroundColor = 'yellow';

Creación y Eliminación de Elementos

Puedes crear nuevos elementos y agregarlos al DOM utilizando métodos como createElement y appendChild. También puedes eliminar elementos utilizando removeChild.

// Crear un nuevo elemento
let nuevoElemento = document.createElement('div');
nuevoElemento.innerHTML = 'Nuevo elemento';
document.body.appendChild(nuevoElemento);

// Eliminar un elemento existente
let elementoAEliminar = document.getElementById('miId');
elementoAEliminar.parentNode.removeChild(elementoAEliminar);

Manejo de Eventos

Puedes agregar manejadores de eventos a los elementos del DOM para responder a acciones del usuario, como clics y pulsaciones de teclas.

let boton = document.getElementById('miBoton');
boton.addEventListener('click', function() {
    alert('Botón clickeado');
});

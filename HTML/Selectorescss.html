Selectores CSS

¿Qué son los selectores de CSS?

Los selectores de CSS son patrones utilizados para seleccionar los elementos a los que se aplicarán las reglas de estilo. A continuación, se presentan los selectores de CSS más comunes:

  • Selector de ID (#id): Selecciona el elemento que tiene el ID especificado.
  • Selector de Clase (.class): Selecciona todos los elementos que tienen la clase especificada.
  • Selector de Elemento (element): Selecciona todos los elementos de un tipo específico.
  • Selector de Clase de Elemento (element.class): Selecciona los elementos de un tipo específico que también tienen una clase específica.
  • Selector de Descendiente (element element): Selecciona los elementos descendientes de otro elemento.
  • Selector de Hijo (element > element): Selecciona los elementos que son hijos directos de un elemento específico.
  • Selector de Hermano Adyacente (element + element): Selecciona el elemento que es adyacente (hermano inmediato) a otro elemento específico.
  • Selector de Hermano General (element ~ element): Selecciona todos los elementos que son hermanos de un elemento específico.

Ejemplos de Selectores CSS

SELECTOR EJEMPLO Descripción
#id #red Selecciona el elemento que tiene el id "red"
.class .general Selecciona todos los elementos que tienen la clase "general"
element h1 Afecta a todos los elementos del tipo especificado, por ejemplo, todos los h1 en el HTML
element.class p.info-general Selecciona todas las etiquetas del tipo especificado que tienen la clase especificada
.class1 .class2 .name1 .name2 Selecciona todos los elementos con la clase "name2" que son hijos directos de un elemento con la clase "name1"
element,element div,p Selecciona todos los elementos del tipo especificado, por ejemplo, todos los divs o todos los párrafos

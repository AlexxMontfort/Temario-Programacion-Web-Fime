Atributos y Clases en jQuery

Manipulación de Atributos en jQuery

jQuery proporciona métodos para obtener, establecer y eliminar atributos de los elementos del DOM de manera fácil y eficiente.

attr()

El método attr() se utiliza para obtener o establecer el valor de uno o más atributos para los elementos seleccionados.

// Obtener el valor del atributo 'href' de un enlace
let href = $("a").attr("href");

// Establecer el atributo 'href' de un enlace
$("a").attr("href", "https://www.example.com");

removeAttr()

El método removeAttr() se utiliza para eliminar un atributo de los elementos seleccionados.

// Eliminar el atributo 'href' de un enlace
$("a").removeAttr("href");

Manipulación de Clases en jQuery

jQuery proporciona métodos para agregar, eliminar, alternar y verificar clases en los elementos del DOM.

addClass()

El método addClass() se utiliza para agregar una o más clases a los elementos seleccionados.

// Agregar la clase 'nuevaClase' a un elemento
$("p").addClass("nuevaClase");

removeClass()

El método removeClass() se utiliza para eliminar una o más clases de los elementos seleccionados.

// Eliminar la clase 'nuevaClase' de un elemento
$("p").removeClass("nuevaClase");

toggleClass()

El método toggleClass() se utiliza para alternar (agregar o eliminar) una clase en los elementos seleccionados. Si la clase está presente, se elimina; si no, se agrega.

// Alternar la clase 'nuevaClase' en un elemento
$("p").toggleClass("nuevaClase");

hasClass()

El método hasClass() se utiliza para verificar si al menos uno de los elementos seleccionados tiene una clase específica.

// Verificar si un elemento tiene la clase 'nuevaClase'
if ($("p").hasClass("nuevaClase")) {
    console.log("El párrafo tiene la clase 'nuevaClase'");
}

Traversing en jQuery

Introducción al Traversing en jQuery

Traversing en jQuery se refiere a la capacidad de navegar a través del DOM para seleccionar elementos en relación con otros elementos. jQuery proporciona una variedad de métodos para moverse hacia arriba, hacia abajo y hacia los lados en el árbol del DOM.

Moverse Hacia Abajo en el DOM

children()

Selecciona todos los elementos hijos directos del elemento seleccionado.

$(selector).children([selectorDeHijos])

Ejemplo: Seleccionar todos los hijos directos de un div.

$("div").children();

find()

Selecciona todos los elementos descendientes del elemento seleccionado que coinciden con el selector especificado.

$(selector).find(selectorDeDescendientes)

Ejemplo: Seleccionar todos los párrafos dentro de un div.

$("div").find("p");

Moverse Hacia Arriba en el DOM

parent()

Selecciona el elemento padre directo del elemento seleccionado.

$(selector).parent()

Ejemplo: Seleccionar el padre directo de un párrafo.

$("p").parent();

parents()

Selecciona todos los elementos ancestros del elemento seleccionado hasta el documento raíz.

$(selector).parents([selectorDeAncestros])

Ejemplo: Seleccionar todos los ancestros de un párrafo.

$("p").parents();

closest()

Selecciona el ancestro más cercano del elemento seleccionado que coincide con el selector especificado.

$(selector).closest(selectorDeAncestros)

Ejemplo: Seleccionar el ancestro más cercano que es un div.

$("p").closest("div");

Moverse Lateralmente en el DOM

siblings()

Selecciona todos los elementos hermanos del elemento seleccionado.

$(selector).siblings([selectorDeHermanos])

Ejemplo: Seleccionar todos los hermanos de un párrafo.

$("p").siblings();

next()

Selecciona el siguiente hermano directo del elemento seleccionado.

$(selector).next([selectorDeSiguiente])

Ejemplo: Seleccionar el siguiente hermano directo de un párrafo.

$("p").next();

prev()

Selecciona el hermano directo anterior del elemento seleccionado.

$(selector).prev([selectorDeAnterior])

Ejemplo: Seleccionar el hermano directo anterior de un párrafo.

$("p").prev();

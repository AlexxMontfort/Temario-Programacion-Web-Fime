Variables en JavaScript

Declaración de Variables

En JavaScript, hay tres formas principales de declarar variables: var, let, y const.

  • var: Permite declarar variables con un alcance global o de función.
  • let: Permite declarar variables con un alcance de bloque.
  • const: Permite declarar variables cuyo valor no puede cambiar una vez asignado y también tiene un alcance de bloque.

Ejemplo de var

var nombre = "Juan";
console.log(nombre); // Imprime: Juan

function saludar() {
    var saludo = "Hola";
    console.log(saludo); // Imprime: Hola
}

saludar();
console.log(saludo); // Error: saludo no está definida

Ejemplo de let

let edad = 25;
console.log(edad); // Imprime: 25

if (true) {
    let edad = 30;
    console.log(edad); // Imprime: 30
}

console.log(edad); // Imprime: 25

Ejemplo de const

const PI = 3.14159;
console.log(PI); // Imprime: 3.14159

// PI = 3.14; // Error: No se puede reasignar un valor a una variable const

if (true) {
    const PI = 3.14;
    console.log(PI); // Imprime: 3.14
}

console.log(PI); // Imprime: 3.14159

Tipos de Datos

Las variables en JavaScript pueden contener valores de diferentes tipos de datos:

  • Números: let numero = 42;
  • Cadenas de Texto (Strings): let texto = "Hola, Mundo!";
  • Booleanos: let esVerdadero = true;
  • Objetos: let persona = { nombre: "Juan", edad: 30 };
  • Arreglos (Arrays): let numeros = [1, 2, 3, 4, 5];
  • Nulo (Null) y No Definido (Undefined): let valorNulo = null;, let valorIndefinido;

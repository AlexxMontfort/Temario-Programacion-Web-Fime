Bucles en JavaScript

Bucle for

El bucle for se utiliza cuando se sabe cuántas veces se debe repetir un bloque de código. Consiste en tres partes: la inicialización, la condición y la actualización.

Inicialización: Se ejecuta una vez al comienzo del bucle. Generalmente se utiliza para declarar e inicializar una variable de control.

Condición: Se evalúa antes de cada iteración. Si la condición es verdadera, el bloque de código dentro del bucle se ejecuta. Si es falsa, el bucle termina.

Actualización: Se ejecuta al final de cada iteración. Generalmente se utiliza para actualizar la variable de control.

for (let i = 0; i < 5; i++) {
    console.log(i);
}
// Imprime: 0, 1, 2, 3, 4

Bucle while

El bucle while ejecuta un bloque de código mientras una condición especificada es verdadera. La condición se evalúa antes de cada iteración.

Se utiliza cuando no se sabe cuántas veces se ejecutará el bucle, pero se sabe que debe continuar hasta que se cumpla una condición específica.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
// Imprime: 0, 1, 2, 3, 4

Bucle do...while

El bucle do...while es similar al while, pero la condición se evalúa después de ejecutar el bloque de código. Esto significa que el bloque de código se ejecuta al menos una vez.

Es útil cuando se necesita que el bloque de código se ejecute al menos una vez, independientemente de si la condición es verdadera o falsa.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
// Imprime: 0, 1, 2, 3, 4

Bucle for...in

El bucle for...in se utiliza para iterar sobre las propiedades enumerables de un objeto. En cada iteración, se asigna una clave de propiedad a la variable especificada.

Es útil para recorrer todas las propiedades de un objeto, pero debe usarse con precaución ya que también itera sobre las propiedades heredadas del prototipo.

let persona = { nombre: "Juan", edad: 30, ciudad: "Madrid" };
for (let propiedad in persona) {
    console.log(propiedad + ": " + persona[propiedad]);
}
// Imprime:
// nombre: Juan
// edad: 30
// ciudad: Madrid

Bucle for...of

El bucle for...of se utiliza para iterar sobre objetos iterables (como arrays, cadenas, etc.). En cada iteración, se asigna el valor del siguiente elemento del iterable a la variable especificada.

Es útil para recorrer elementos de arrays o cualquier objeto iterable de manera más directa y limpia que usando un índice.

let numeros = [1, 2, 3, 4, 5];
for (let numero of numeros) {
    console.log(numero);
}
// Imprime: 1, 2, 3, 4, 5

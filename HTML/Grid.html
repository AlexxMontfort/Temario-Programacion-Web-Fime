Sistema de Grid en Bootstrap

¿Qué es el Sistema de Grid?

El sistema de grid (rejilla) de Bootstrap es una de sus características más poderosas y populares. Este sistema permite crear diseños responsivos y adaptativos para todo tipo de dispositivos y tamaños de pantalla de manera sencilla y eficiente. La grid de Bootstrap se basa en flexbox, lo que proporciona un control preciso sobre el diseño de la página.

El sistema de grid de Bootstrap está compuesto por contenedores (containers), filas (rows) y columnas (columns):

  • Contenedores (`.container` o `.container-fluid`): Estos encapsulan filas y columnas para centrar y alinear el contenido de manera apropiada.
  • Filas (`.row`): Las filas sirven como contenedores horizontales para las columnas.
  • Columnas (`.col`): Las columnas se anidan dentro de las filas y utilizan clases predefinidas de Bootstrap para definir su ancho.

Clases de Columna

Bootstrap divide el espacio disponible en 12 columnas. Utilizando diferentes clases, puedes especificar cuántas de estas columnas debe ocupar un elemento. Las clases de columna son responsivas y se ajustan a diferentes tamaños de pantalla usando los puntos de quiebre:

  • .col-: Para dispositivos extra pequeños (menos de 576px)
  • .col-sm-: Para dispositivos pequeños (576px y más)
  • .col-md-: Para dispositivos medianos (768px y más)
  • .col-lg-: Para dispositivos grandes (992px y más)
  • .col-xl-: Para dispositivos extra grandes (1200px y más)

Ejemplo de Grid Básico

<div class="container">
    <div class="row">
        <div class="col">.col</div>
        <div class="col">.col</div>
        <div class="col">.col</div>
    </div>
</div>

En este ejemplo, cada columna ocupa una parte igual del espacio disponible dentro de la fila.

Columnas con Diferente Tamaño

Es posible definir columnas con diferentes anchos utilizando las clases apropiadas de Bootstrap:

<div class="container">
    <div class="row">
        <div class="col-6">.col-6</div>
        <div class="col-6">.col-6</div>
    </div>
    <div class="row mt-3">
        <div class="col-4">.col-4</div>
        <div class="col-4">.col-4</div>
        <div class="col-4">.col-4</div>
    </div>
</div>

En este ejemplo, la primera fila tiene dos columnas que ocupan el 50% del ancho cada una, y la segunda fila tiene tres columnas que ocupan el 33.33% del ancho cada una.

Grid Responsivo

Bootstrap permite crear diseños responsivos que se adaptan a diferentes tamaños de pantalla utilizando clases de columnas específicas para cada punto de quiebre:

<div class="container">
    <div class="row">
        <div class="col-12 col-sm-6 col-md-4">.col-12 .col-sm-6 .col-md-4</div>
        <div class="col-12 col-sm-6 col-md-4">.col-12 .col-sm-6 .col-md-4</div>
        <div class="col-12 col-sm-6 col-md-4">.col-12 .col-sm-6 .col-md-4</div>
    </div>
</div>

En este ejemplo, las columnas ocupan el 100% del ancho en dispositivos extra pequeños, el 50% en dispositivos pequeños, y el 33.33% en dispositivos medianos y más grandes.

.col-12 .col-sm-6 .col-md-4
.col-12 .col-sm-6 .col-md-4
.col-12 .col-sm-6 .col-md-4

Offsetting Columns

Es posible desplazar columnas utilizando clases de offset. Esto puede ser útil para centrar contenido o crear espacios vacíos entre columnas:

<div class="container">
    <div class="row">
        <div class="col-md-4">.col-md-4</div>
        <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
    </div>
    <div class="row mt-3">
        <div class="col-md-3 offset-md-3">.col-md-3 .offset-md-3</div>
        <div class="col-md-3 offset-md-3">.col-md-3 .offset-md-3</div>
    </div>
</div>

En este ejemplo, la primera fila tiene una columna de 4 unidades y otra columna de 4 unidades desplazada 4 unidades a la derecha. La segunda fila tiene dos columnas de 3 unidades desplazadas 3 unidades a la derecha.

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3

Etiquetas HTML Fundamentales

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. A continuación, se describen algunas de las etiquetas HTML más fundamentales, junto con su propósito y uso.

<html>

Define el inicio y el final de un documento HTML.

<head>

Contiene meta-información sobre el documento, como su título y enlaces a hojas de estilo y scripts.

<title>

Especifica el título del documento que aparece en la barra de título del navegador.

<body>

Define el cuerpo del documento, que contiene el contenido visible para los usuarios.

<h1> - <h6>

Define encabezados, donde <h1> es el más importante y <h6> es el menos importante.

<p>

Define un párrafo de texto.

<br>

Inserta un salto de línea.

<hr>

Inserta una línea horizontal.

<strong>

Define texto importante (negrita).

<em>

Define texto enfatizado (cursiva).

<a>

Define un hipervínculo, que se utiliza para enlazar a otras páginas o recursos.

<img>

Define una imagen.

<ul>, <ol>, <li>

Define listas no ordenadas (<ul>), listas ordenadas (<ol>), y elementos de lista (<li>).

<table>, <tr>, <td>, <th>

Define una tabla (<table>), filas de tabla (<tr>), celdas de tabla (<td>), y celdas de encabezado de tabla (<th>).

<form>, <input>, <textarea>, <button>, <label>

Define un formulario (<form>), campos de entrada (<input>), áreas de texto (<textarea>), botones clicables (<button>), y etiquetas para elementos de formulario (<label>).

<audio>, <video>, <source>

Define contenido de audio (<audio>), contenido de video (<video>), y múltiples recursos de medios (<source>).

<div>, <span>

Define una división o sección en un documento (<div>) y una sección en línea dentro de un documento (<span>).

<link>

Define la relación entre el documento actual y un recurso externo.

<meta>

Define metadatos sobre un documento HTML.

<script>

Define un script de cliente (JavaScript).

<style>

Define estilos CSS.

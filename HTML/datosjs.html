Tipos de Datos en JavaScript

Tipos de Datos Primitivos

JavaScript tiene varios tipos de datos primitivos:

  • Número (`Number`)
  • Cadena de Texto (`String`)
  • Booleano (`Boolean`)
  • Indefinido (`Undefined`)
  • Nulo (`Null`)
  • Símbolo (`Symbol`)
  • BigInt (`BigInt`)

Número (`Number`)

let entero = 42;
let decimal = 3.14;

Cadena de Texto (`String`)

let cadena1 = "Hola, Mundo!";
let cadena2 = 'JavaScript es genial';
let cadena3 = `Esto es una cadena con comillas invertidas`;

Booleano (`Boolean`)

let verdadero = true;
let falso = false;

Indefinido (`Undefined`)

let indefinido;
console.log(indefinido); // Imprime: undefined

Nulo (`Null`)

let nulo = null;

Símbolo (`Symbol`)

let simbolo1 = Symbol('descripcion');
let simbolo2 = Symbol('descripcion');
console.log(simbolo1 === simbolo2); // Imprime: false

BigInt (`BigInt`)

let numeroGrande = BigInt(1234567890123456789012345678901234567890);

Tipos de Datos de Objeto

JavaScript también tiene tipos de datos más complejos llamados objetos:

  • Objeto (`Object`)
  • Arreglo (`Array`)
  • Fecha (`Date`)
  • Expresión Regular (`RegExp`)
  • Función (`Function`)

Objeto (`Object`)

let persona = {
    nombre: "Juan",
    edad: 30,
    saludar: function() {
        console.log("Hola, me llamo " + this.nombre);
    }
};

Arreglo (`Array`)

let numeros = [1, 2, 3, 4, 5];
let mezclado = [1, 'dos', true, { clave: 'valor' }, [6, 7, 8]];

Fecha (`Date`)

let fechaActual = new Date();
console.log(fechaActual);

Expresión Regular (`RegExp`)

let expresion = /hola/;
let cadena = "hola mundo";
console.log(expresion.test(cadena)); // Imprime: true

Función (`Function`)

function saludar(nombre) {
    return "Hola, " + nombre;
}

console.log(saludar("Mundo")); // Imprime: Hola, Mundo

Manejo de Eventos en jQuery

Introducción al Manejo de Eventos en jQuery

El manejo de eventos en jQuery facilita la vinculación de manejadores de eventos a los elementos del DOM. Los eventos son acciones que ocurren en el navegador, como clics del ratón, pulsaciones de teclas o desplazamientos de la página.

Métodos Generales de Manejo de Eventos

on()

El método on() se utiliza para vincular uno o más eventos a los elementos seleccionados.

$(selector).on(event, childSelector, data, function)

Ejemplo: Vincular un evento de clic a todos los párrafos.

$("p").on("click", function() {
    alert("Párrafo clickeado");
});

off()

El método off() se utiliza para eliminar uno o más manejadores de eventos de los elementos seleccionados.

$(selector).off(event, childSelector, function)

Ejemplo: Eliminar el evento de clic de todos los párrafos.

$("p").off("click");

Métodos Específicos de Eventos

click()

Vincula un manejador de eventos de clic a los elementos seleccionados.

$(selector).click(function)

Ejemplo: Vincular un evento de clic a todos los botones.

$("button").click(function() {
    alert("Botón clickeado");
});

dblclick()

Vincula un manejador de eventos de doble clic a los elementos seleccionados.

$(selector).dblclick(function)

Ejemplo: Vincular un evento de doble clic a todos los párrafos.

$("p").dblclick(function() {
    alert("Párrafo doblemente clickeado");
});

mouseenter() y mouseleave()

Vinculan manejadores de eventos para cuando el ratón entra y sale de un elemento, respectivamente.

$(selector).mouseenter(function)
$(selector).mouseleave(function)

Ejemplo: Vincular eventos de mouseenter y mouseleave a un div.

$("div").mouseenter(function() {
    $(this).css("background-color", "yellow");
}).mouseleave(function() {
    $(this).css("background-color", "white");
});

keydown() y keyup()

Vinculan manejadores de eventos para cuando una tecla es presionada y soltada, respectivamente.

$(selector).keydown(function)
$(selector).keyup(function)

Ejemplo: Vincular eventos de keydown y keyup a un campo de texto.

$("input").keydown(function() {
    $(this).css("background-color", "lightblue");
}).keyup(function() {
    $(this).css("background-color", "white");
});

submit()

Vincula un manejador de eventos de envío de formulario a los elementos seleccionados.

$(selector).submit(function)

Ejemplo: Vincular un evento de envío a un formulario.

$("form").submit(function(event) {
    alert("Formulario enviado");
    event.preventDefault(); // Prevenir el envío del formulario
});

change()

Vincula un manejador de eventos de cambio a los elementos seleccionados (usualmente campos de formulario).

$(selector).change(function)

Ejemplo: Vincular un evento de cambio a un campo de selección.

$("select").change(function() {
    alert("Selección cambiada a " + $(this).val());
});

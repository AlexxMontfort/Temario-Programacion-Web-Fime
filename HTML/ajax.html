AJAX con jQuery

Introducción a AJAX

AJAX (Asynchronous JavaScript and XML) es una técnica de desarrollo web que permite a las aplicaciones web comunicarse con el servidor de manera asíncrona, sin necesidad de recargar la página completa. Esto mejora la experiencia del usuario al hacer que las aplicaciones sean más rápidas y dinámicas.

Métodos AJAX en jQuery

$.ajax()

El método $.ajax() es el más versátil y permite un control completo sobre las solicitudes AJAX.

$.ajax({
    url: "https://api.example.com/datos",
    method: "GET",
    dataType: "json",
    success: function(response) {
        console.log(response);
    },
    error: function(xhr, status, error) {
        console.error("Error: " + error);
    }
});

$.get()

El método $.get() es un atajo para realizar solicitudes GET.

$.get("https://api.example.com/datos", function(response) {
    console.log(response);
});

$.post()

El método $.post() es un atajo para realizar solicitudes POST.

$.post("https://api.example.com/datos", { nombre: "Juan", edad: 30 }, function(response) {
    console.log(response);
});

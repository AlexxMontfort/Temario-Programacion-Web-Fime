Selectores de jQuery

Introducción a los Selectores de jQuery

jQuery proporciona una manera fácil y eficiente de seleccionar y manipular elementos del DOM. Los selectores de jQuery permiten encontrar elementos basados en sus etiquetas, clases, IDs, atributos, y más.

Selector de Etiqueta

Selecciona todos los elementos de un tipo específico de etiqueta. Utiliza el nombre de la etiqueta como selector.

$("etiqueta")

Ejemplo: Seleccionar todos los párrafos <p>.

$("p") // Selecciona todos los párrafos

Selector de Clase

Selecciona todos los elementos con una clase específica. Utiliza un punto . seguido del nombre de la clase.

$(".clase")

Ejemplo: Seleccionar todos los elementos con la clase miClase.

$(".miClase") // Selecciona todos los elementos con la clase "miClase"

Selector de ID

Selecciona un elemento con un ID específico. Utiliza una almohadilla # seguida del valor del ID.

$("#id")

Ejemplo: Seleccionar el elemento con el ID miId.

$("#miId") // Selecciona el elemento con el ID "miId"

Selector de Atributo

Selecciona todos los elementos con un atributo específico. Utiliza corchetes [] alrededor del nombre del atributo.

$("[atributo]")

Ejemplo: Seleccionar todos los elementos con el atributo href.

$("[href]") // Selecciona todos los elementos con el atributo "href"

Selector de Pseudo-clase

Selecciona elementos basados en su estado o posición en el DOM. Utiliza dos puntos : seguido del nombre de la pseudo-clase.

$("selector:pseudo-clase")

Ejemplo: Seleccionar el primer párrafo.

$("p:first") // Selecciona el primer párrafo

Selector de Descendiente

Selecciona todos los elementos descendientes de un elemento específico. Utiliza un espacio entre los selectores.

$("ancestro descendiente")

Ejemplo: Seleccionar todos los párrafos dentro de divs.

$("div p") // Selecciona todos los párrafos dentro de divs

Selector de Hijo

Selecciona todos los elementos que son hijos directos de un elemento específico. Utiliza el signo mayor que > entre los selectores.

$("padre > hijo")

Ejemplo: Seleccionar todos los párrafos que son hijos directos de divs.

$("div > p") // Selecciona todos los párrafos que son hijos directos de divs

Selector de Hermano

Selecciona los elementos hermanos de un elemento específico. Utiliza el signo más + para seleccionar el hermano inmediato o la tilde ~ para seleccionar todos los hermanos.

$("elemento + hermano")

Ejemplo: Seleccionar el párrafo que sigue inmediatamente a un h2.

$("h2 + p") // Selecciona el párrafo que sigue inmediatamente a un h2

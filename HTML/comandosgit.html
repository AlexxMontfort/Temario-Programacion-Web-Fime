Comandos de Git

Inicialización y Configuración

git init

Inicializa un nuevo repositorio de Git en el directorio actual.

git init

git config

Configura las opciones de Git, como el nombre del usuario y el correo electrónico.

git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@example.com"

Comandos Básicos

git clone

Clona un repositorio remoto en tu máquina local.

git clone https://github.com/usuario/repositorio.git

git add

Agrega archivos al área de preparación (staging area).

git add nombre_del_archivo
# Para agregar todos los archivos cambiados
git add .

git commit

Crea un commit con los cambios agregados al área de preparación.

git commit -m "Descripción de los cambios"

git status

Muestra el estado de los archivos en el repositorio, indicando los cambios que han sido preparados, los que no han sido preparados y los archivos que no están siendo seguidos por Git.

git status

git log

Muestra el historial de commits del repositorio.

git log

Trabajando con Ramas

git branch

Lista todas las ramas en el repositorio o crea una nueva rama.

# Listar todas las ramas
git branch
# Crear una nueva rama
git branch nombre_de_la_rama

git checkout

Cambia a una rama existente o crea una nueva y cambia a ella.

# Cambiar a una rama existente
git checkout nombre_de_la_rama
# Crear y cambiar a una nueva rama
git checkout -b nombre_de_la_rama

git merge

Combina cambios de una rama en la rama actual.

git merge nombre_de_la_rama

Sincronización con Repositorios Remotos

git remote

Administra las conexiones a repositorios remotos.

# Ver los remotos configurados
git remote -v
# Agregar un nuevo remoto
git remote add origin https://github.com/usuario/repositorio.git

git fetch

Descarga los cambios del repositorio remoto pero no los integra en la rama actual.

git fetch

git pull

Descarga los cambios del repositorio remoto y los fusiona con la rama actual.

git pull origin nombre_de_la_rama

git push

Envía los commits locales al repositorio remoto.

git push origin nombre_de_la_rama

Otros Comandos Útiles

git diff

Muestra las diferencias entre el área de trabajo y el área de preparación o entre commits.

# Diferencias entre el área de trabajo y el área de preparación
git diff
# Diferencias entre el área de preparación y el último commit
git diff --staged

git reset

Deshace cambios en el área de preparación o en el historial de commits.

# Deshacer cambios en el área de preparación
git reset nombre_del_archivo
# Deshacer el último commit (manteniendo los cambios en el área de trabajo)
git reset --soft HEAD~1
# Deshacer el último commit (perdiendo los cambios)
git reset --hard HEAD~1

git rm

Elimina archivos del repositorio y del área de trabajo.

git rm nombre_del_archivo

git stash

Guarda los cambios no commitados temporalmente y limpia el área de trabajo.

git stash
# Recuperar los cambios guardados
git stash pop

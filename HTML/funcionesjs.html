Funciones en JavaScript

Declaración de Funciones

Las funciones en JavaScript pueden ser declaradas de varias formas:

  • Declaración de Función
  • Expresión de Función
  • Funciones Flecha

Declaración de Función

function saludar(nombre) {
    return "Hola, " + nombre;
}

console.log(saludar("Juan")); // Imprime: Hola, Juan

Expresión de Función

let saludar = function(nombre) {
    return "Hola, " + nombre;
};

console.log(saludar("María")); // Imprime: Hola, María

Funciones Flecha

let saludar = (nombre) => {
    return "Hola, " + nombre;
};

console.log(saludar("Carlos")); // Imprime: Hola, Carlos
let saludar = nombre => "Hola, " + nombre;

console.log(saludar("Ana")); // Imprime: Hola, Ana

Parámetros y Argumentos

Las funciones pueden aceptar parámetros, que son valores que se pasan a la función cuando se llama. Los argumentos son los valores reales que se pasan a la función.

function sumar(a, b) {
    return a + b;
}

console.log(sumar(2, 3)); // Imprime: 5

Valores por Defecto de los Parámetros

Se pueden definir valores por defecto para los parámetros, que serán utilizados si no se proporciona un valor al llamar a la función.

function saludar(nombre = "Invitado") {
    return "Hola, " + nombre;
}

console.log(saludar()); // Imprime: Hola, Invitado
console.log(saludar("Laura")); // Imprime: Hola, Laura

Funciones Anidadas

Las funciones pueden ser definidas dentro de otras funciones. Las funciones internas tienen acceso a las variables y parámetros de las funciones externas.

function externa() {
    let mensaje = "Hola desde la función externa";

    function interna() {
        console.log(mensaje);
    }

    interna();
}

externa(); // Imprime: Hola desde la función externa
